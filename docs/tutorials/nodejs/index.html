        # Setup a new project

        # Prerequisites

        This assumes you have installed node.js on your system. Check by using the following command in a terminal.

        node -v

        If you see command not found: node then node.js is missing. Install it before proceeding. Node.js comes with npm, package manager for JavaScript. Another alternative is yarn. To check you npm installed, run the following command in a terminal.

        npm -v

        # Initialize node project

        • First lets go to your tutorial directory.
        cd ~/tutorials
        • Create a new directory for our node.js project and enter it.
        mkdir node-init && cd node-init
        • Create a new package.json for our project.
        npm init -y

        TIP

        -y is a npm flag which just defaults all the answers to Y or yes in npm init prompts. The package.json file can be edited later on. This allows us to quickly set up. More info about package.json here.

        # Creating first file

        Now that we have our initial setup complete, we can finally start writing code. At this point, it's wise to open this folder in your code editor. The folder should contain just one single file, package.json which we created in the previous step.

        • Create a new file in root.
        touch index.js
        • Write some JavaScript in it.
        console.log('Hello world!');
        • Save the file.
        • Execute the file.
        node index.js
        • You should see the following output:
        Hello world!
