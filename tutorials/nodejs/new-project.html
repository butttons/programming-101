      # New nodejs project

      # Initialize node project

      • First lets go to your tutorial directory.
      cd ~/tutorials
      • Create a new directory for our node.js project and enter it.
      mkdir node-init && cd node-init
      • Create a new package.json for our project.
      npm init -y

      TIP

      -y is a npm flag which just defaults all the answers to Y or yes in npm init prompts. The package.json file can be edited later on. This allows us to quickly set up. More info about package.json here.

      # Creating first file

      This is completely optional. Now that we have our initial setup complete, we can finally start writing code. At this point, it's wise to open this folder in your code editor. The folder should contain just one single file, package.json which we created in the previous step.

      • Create a new file in root.
      touch index.js
      • Write some JavaScript in it.
      console.log('Hello world!');
      • Save the file.
      • Execute the file.
      node index.js
      • You should see the following output:
      Hello world!
